# Zenodo Release Notes R19

Suggested release tag: `v1.0.0-itcon-r19`

## Purpose

This release archives the public reproducibility package for the ITcon-aligned RuleBridge-BIM R19 manuscript:

**An OpenBIM evidence-readiness diagnostic for pre-compliance information governance: The RuleBridge-BIM protocol**

The release supports a reproducible OpenBIM evidence-readiness diagnostic rather than a legal compliance decision system. It includes scripts, configurations, derived tables, generated figures, source registries and reference-check outputs needed to inspect the reported public-corpus results.

## Main Contents

- Expanded public IFC discovery and corpus stratification outputs.
- Deterministic RuleBridge-BIM evidence-readiness checks.
- IfcOpenShell direct-field reference outputs.
- IDS/IfcTester reference outputs for the IDS-expressible subset of checks.
- Evidence-route graph summaries and project-delivery triage outputs.
- Evidence-mutation and synthetic-repair sensitivity outputs.
- Generated reporting figures and tables.
